# TEMP — 臨時雜項工具

各種臨時使用的小工具腳本集合，功能各自獨立。

## 工具清單

| 檔案 | 功能 |
|------|------|
| `rdc.py` | 解析 Verilog，在時間延遲（`#100ns`）與 wait/check task 前後加入 display 語句 |
| `replace_inst_name.py` | 依設定檔批次替換 Verilog 中的 instance 名稱 |
| `all_module_prefix.py` | 批次為 Verilog module 名稱加上前綴 |
| `all_filename_prefix.py` | 批次為檔案名稱加上前綴 |
| `all_dict_rename.py` | 依字典對應批次重命名 |
| `check_dic_qos.py` | 檢查 QoS 字典設定 |
| `check_warnin.py` | 解析並過濾 warning 訊息 |
| `config_set_add_order.py` | 設定檔順序新增工具 |
| `proj_data.py` | 專案資料管理 |
| `tbl_formate_chg_for_parser.py` | 轉換 table 格式以供 parser 使用 |
| `tchek_diff.py` | Timing check 差異比對 |
| `temp.py` | 臨時測試腳本 |
| `record_voice.py` | 錄音小工具 |
